# OVD Eyewear Image Placeholder Map

This file is the master image plan for the website.
Use it to assign image-generation/download tasks to agents.

## Rules
- Save all final assets inside the images folder.
- Keep naming exactly as listed for predictable integration.
- Preferred formats: WebP or PNG for transparent assets, JPG/WebP for photos.
- Suggested minimum sizes:
  - Hero and banners: 1920x1080
  - Section cards: 1200x900
  - Product cards: 1200x1200
  - Logos/icons: SVG or 1024x1024 PNG

## Status Key
- TODO: not sourced yet
- READY: image is finalized and available
- REPLACE: temporary image exists, final one pending
